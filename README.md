# 📊 Market Analyzer Dashboard

A professional GUI-based Python application to analyze and visualize financial data from **Stock Market** and **Cryptocurrency (Bitcoin)** datasets.

---

## 🚀 Features

* 📈 Visualize price trends using interactive graphs
* 📊 Display summary statistics (Max, Min, Average)
* 📉 Calculate Moving Average for trend analysis
* 🔄 Compare Stock vs Bitcoin in a single graph
* 🖥️ Modern GUI dashboard using Tkinter

---

## 🛠️ Technologies Used

* **Python**
* **Tkinter** – GUI Development
* **Pandas** – Data Processing
* **Matplotlib** – Data Visualization
* **OOP Concepts** – Modular Design
